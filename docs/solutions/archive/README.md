# Archived Solution Notes

This directory is for resolved notes that no longer need to live in the active
`docs/solutions/` tree.

Move a note here when it is primarily historical:

- the bug is fixed
- regression coverage exists
- the note no longer changes day-to-day implementation choices

Archive notes should still be readable, but they do not need to follow the same
level of curation as active `implementation/`, `testing/`, or `logic-errors`
notes.
